# Root-Checker
Display all root related info of an Android Device. 


### Usage:
1. Open project and sync gradle.
2. Modify the code to suit your needs.
3. Watch out for more features.


&nbsp;
### Stats Displayed :
- Device Rooted
- Root Available
- Root Path
- Root given
- Busy Box installation.
